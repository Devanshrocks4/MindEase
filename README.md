# 🌌 Overview

MindEase is a **full-stack AI-powered mental wellness platform** designed to deliver
**accessible, intelligent, and real-time psychological support**.

It integrates:

* 🧪 Clinical Assessments
* 🤖 AI Therapy (Gemini)
* 📊 Data Analytics
* ⚡ Real-Time Systems
